How It Works
Everything about your Second Brain.

What is it?

An AI assistant that remembers you, your projects, and your work. Uses Claude Code with PARA to organise knowledge and automate tasks.

Ask My Brain

Type any task in plain language. Your brain runs it. No commands needed.

How does it remember?

Stores info in CLAUDE.md. About Me and routines auto-sync. Every session reads it first.

Routines

Plain language instruction sets. Click Run and it executes each step.

Data safety

Everything runs within Meta infrastructure. Nothing leaves the network.
