My Second Brain

A web dashboard for Claude Code. Connect to your server, open the dashboard, and tell your brain what to do — in plain language. Minimal terminal, maximum output.

Get Started

My Second Brain runs on a remote server — either an OD (On-Demand, temporary) or a Dev Server (permanent, reserved). Not sure which you have? Ask your team.

🚀

First Time Setup

New here? One-time setup takes about 5 minutes. Same steps for Mac and Windows.

Setup Guide →
☁️

Log in via OD

1. Open a terminal and run:

dev connect Click to copy

2. Update & start the server:

cd ~/my-second-brain && git pull && bash setup.sh Click to copy

3. Open localhost:5151 in your browser

🖥️

Log in via Dev Server

1. Open a terminal and SSH into your server:

ssh -L 5151:localhost:5151 your-devserver-name Click to copy

2. Update & start the server:

cd ~/my-second-brain && git pull && bash setup.sh Click to copy

3. Open localhost:5151 in your browser

The server must be running for the dashboard to work. If you close the terminal, the dashboard stops.

What it does

🧠

Connect Brain

One click to connect to Claude Code. No terminal needed after first time setup.

💬

Ask My Brain

Type any task in plain language. "Prep my 2pm meeting", "Summarize my tasks". Your brain handles it.

⚡

Routines

Morning brief and EOD debrief, one click each. Add your own steps in plain language.

👤

About Me

Fill in your details once. Auto-syncs to CLAUDE.md so your brain always knows who you are.

🎙️

Talk to Me

Voice input mode — speak your tasks instead of typing. Coming soon.
