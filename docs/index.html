My Second Brain

A web dashboard for Claude Code that removes the terminal barrier. Open a page, connect your brain, tell it what to do. No commands, no code.

Get Started View on GitHub

What it does

🧠

Connect Brain

One click to connect to Claude Code on your OD. No terminal needed after setup.

🔨

Build Now

Type any task in plain language. "Prep my 2pm meeting", "Summarize my tasks". Your brain handles it.

⚡

Routines

Morning brief and EOD debrief, one click each. Add your own steps in plain language.

👤

About Me

Fill in your details once. Auto-syncs to CLAUDE.md so your brain always knows who you are.

🎙️

Talk to Me

Voice input mode — speak your tasks instead of typing. Coming soon.

📖

Built-in Guide

Get Started page, How It Works, and troubleshooting — all inside the app.

Setup in 60 seconds

One command. Then it's all browser-based.

# Clone and start your Second Brain
git clone https://github.com/robglre/my-second-brain.git
cd my-second-brain
bash setup.sh

Then open http://localhost:5151 and click "Connect Brain".
